PTP Processing of supplier accounts is prevented unless all mandatory fields are complete
To ensure that vendor master information is complete within the SAP system, key fields in the vendor master should be set to mandatory. This field control is arranged via vendor accounts groups; there are multiple vendor groups are used within MARC. When a mandatory field is not used an error message will pop up. This control therefore prevents procesing of incomplete master data.
